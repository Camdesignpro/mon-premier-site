Pourquoi cette structure marche bien

Les couleurs, l'espacement et les rayons sont pilotés par des variables, ce qui simplifie les futures évolutions.

Primitives

Les valeurs brutes : bleu, gris, blanc, noir. Elles servent de base technique.

Semantic

Les alias métier : brand, text, background, border. Elles portent le sens.

Mapped

Les usages concrets : surface, icon, text, border. Elles parlent aux composants.
